# Gemini Models (Project Policy Summary)

Canonical policy: read `GEMINI_3_ONLY.md`.

This project only allows **Gemini 3** models for frontend work:

- Default: `gemini-3-flash-preview`
- Fallback: `gemini-3-pro-preview` (use only when Flash is insufficient or you hit constraints)

Not allowed:
- Any Gemini 2.x models (for example `gemini-2.5-*`)
